XML IDs dalam Odoo memiliki format 'module.identifier', misalnya 'export.product_product_14828_24ea7638'.
- 'export' adalah nama modul
- 'product_product_14828_24ea7638' adalah identifier unik untuk record tersebut

record secara unik di seluruh database, terutama untuk proses impor/ekspor data.

================================================================================
SPECIAL SECTION: PRODUCT SUPPLIER INFO RELATIONSHIPS
================================================================================

Hubungan antar model untuk informasi supplier produk:

1. product.product (memiliki field default_code = 'Internal References')
   ↓ (produk memiliki product_tmpl_id yang mengarah ke template)
2. product.template (template produk)
   ↓ (template memiliki seller_ids yang mengarah ke supplier info)
3. product.supplierinfo (info supplier untuk produk)
   - Field name: mengarah ke res.partner (vendor)
   - Field product_tmpl_id: mengarah ke product.template
   - Field product_id: mengarah ke product.product (opsional, untuk variant spesifik)
   - Field price: harga dari supplier

CONTOH CARA MENGAMBIL DATA SUPPLIER UNTUK PRODUK:

1. Cari product.product berdasarkan default_code ('Internal References')
2. Dapatkan product_tmpl_id-nya
3. Cari di product.supplierinfo berdasarkan product_tmpl_id
4. Filter berdasarkan nama supplier jika diperlukan

CONTOH KODE PYTHON:

```python
# Cari produk berdasarkan default_code
product_ids = models.execute_kw(
    db, uid, password, 'product.product', 'search',
    [[['default_code', '=', 'KODE-PRODUK']]]
)

# Dapatkan template_id
product_data = models.execute_kw(
    db, uid, password, 'product.product', 'read',
    [product_ids[0]], {'fields': ['product_tmpl_id']}
)
product_tmpl_id = product_data[0]['product_tmpl_id'][0]

# Cari supplier info
supplier_info = models.execute_kw(
    db, uid, password, 'product.supplierinfo', 'search_read',
    [[['product_tmpl_id', '=', product_tmpl_id]]],
    {'fields': ['name', 'price']}
)

# Dapatkan nama supplier
for info in supplier_info:
    partner_id = info['name'][0]
    partner_data = models.execute_kw(
        db, uid, password, 'res.partner', 'read',
        [partner_id], {'fields': ['name']}
    )
    info['supplier_name'] = partner_data[0]['name']
```
